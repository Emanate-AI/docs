# API Endpoints

Complete reference for all Emanate API endpoints.

## Leads

### List Leads

```bash
GET /v1/leads
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| status | string | Filter by status |
| source | string | Filter by source |
| minScore | number | Minimum ICP score |
| limit | number | Results per page |
| offset | number | Pagination offset |

**Response:**

```json
{
  "success": true,
  "data": [
    {
      "id": "lead_123",
      "name": "John Smith",
      "email": "john@example.com",
      "company": "Acme Corp",
      "status": "qualified",
      "icpScore": 85,
      "source": "voice",
      "createdAt": "2024-02-04T10:00:00Z"
    }
  ],
  "pagination": {
    "total": 150,
    "limit": 50,
    "offset": 0
  }
}
```

### Get Lead

```

**Response:**

{

  "data": {
    "id": "lead_123",
    "name": "John Smith",
    "email": "john@example.com",
    "phone": "+14155551234",
    "company": "Acme Corp",
    "title": "VP Operations",
    "status": "qualified",
    "icpScore": 85,
    "enrichment": {
      "companySize": "500-1000",
      "industry": "Manufacturing",
      "linkedIn": "linkedin.com/in/johnsmith"
    },
    "source": "voice",
    "agentId": "asst_456",
    "conversationId": "call_789",
    "crmId": "hubspot_123",
    "createdAt": "2024-02-04T10:00:00Z",
    "updatedAt": "2024-02-04T10:05:00Z"
  }

```

### Create Lead

```bash
POST /v1/leads
```

**Request Body:**

```json
{
  "name": "Jane Doe",
  "email": "jane@example.com",
  "company": "Widget Inc",
  "phone": "+14155555678",
  "source": "api",
  "customFields": {
    "projectType": "construction",
    "timeline": "Q2 2024"
  }
}
```

### Update Lead

```bash
PUT /v1/leads/:id
```

**Request Body:**

```json
{
  "status": "contacted",
  "notes": "Follow-up call scheduled"
}
```

### Delete Lead

```bash
DELETE /v1/leads/:id
```

## Agents

### List Agents

```bash
GET /v1/agents
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| type | string | voice, chat |
| status | string | active, paused |

**Response:**

```json
{
  "success": true,
  "data": [
    {
      "id": "asst_123",
      "name": "Sales Assistant",
      "type": "voice",
      "status": "active",
      "createdAt": "2024-01-15T08:00:00Z"
    }
  ]
}
```

### Get Agent

```bash
GET /v1/agents/:id
```

### Create Agent

```bash
POST /v1/agents
```

**Request Body:**

```json
{
  "name": "New Agent",
  "type": "voice",
  "firstMessage": "Hi, how can I help?",
  "systemPrompt": "You are a helpful assistant...",
  "model": {
    "provider": "openai",
    "model": "gpt-4o"
  }
}
```

### Update Agent

```bash
PUT /v1/agents/:id
```

## Campaigns

### List Campaigns

```bash
GET /v1/campaigns
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| status | string | draft, running, paused, completed |

### Get Campaign

```bash
GET /v1/campaigns/:id
```

### Create Campaign

```bash
POST /v1/campaigns
```

**Request Body:**

```json
{
  "name": "Q1 Outreach",
  "agentId": "asst_123",
  "startDate": "2024-02-01",
  "endDate": "2024-02-28",
  "schedule": {
    "monday": { "start": "09:00", "end": "17:00" },
    "tuesday": { "start": "09:00", "end": "17:00" }
  },
  "pacing": {
    "callsPerHour": 20,
    "maxAttempts": 3
  }
}
```

### Start Campaign

```bash
POST /v1/campaigns/:id/start
```

### Pause Campaign

```bash
POST /v1/campaigns/:id/pause
```

### Add Contacts

```bash
POST /v1/campaigns/:id/contacts
```

**Request Body:**

```json
{
  "contacts": [
    {
      "phone": "+14155551234",
      "firstName": "John",
      "lastName": "Smith",
      "company": "Acme"
    }
  ]
}
```

## Error Codes

| Code | Description |
|------|-------------|
| `INVALID_REQUEST` | Malformed request |
| `UNAUTHORIZED` | Invalid API key |
| `FORBIDDEN` | Insufficient permissions |
| `NOT_FOUND` | Resource not found |
| `RATE_LIMITED` | Too many requests |
| `INTERNAL_ERROR` | Server error |
