# Chat Logs

Chat logs provide complete history of all chat widget conversations, including messages, metadata, and lead information.

## Accessing Chat Logs

Navigate to **Analytics** → **Chat Logs**

## Session List

The session list shows all chat conversations:

| Column | Description |
|--------|-------------|
| **Date/Time** | When session started |
| **Visitor** | Name or ID |
| **Agent** | Chat agent used |
| **Messages** | Message count |
| **Duration** | Session length |
| **Lead** | Lead captured |

### Filtering

Filter sessions by:

- Date range
- Agent
- Has lead
- Message count
- Duration

## Session Detail View

Click a session to see full details:

### Messages

Full chat history:

```
Agent (10:32 AM):
  Hi! How can I help you today?

Visitor (10:32 AM):
  I'm looking for pricing on your steel products

Agent (10:33 AM):
  I'd be happy to help! We offer a wide range of steel products.
  What specific type are you interested in?
  - Structural steel
  - Sheet steel  
  - Steel tubing

Visitor (10:33 AM):
  Structural steel beams
```

### Session Info

- Session ID
- Start/end time
- Page URL where chat started
- Visitor location
- Device type

### Lead Info

If captured:
- Contact details
- Enrichment data
- ICP score
