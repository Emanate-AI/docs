# Campaigns Overview

Campaigns enable automated outbound voice calling at scale. Reach prospects, follow up with leads, and drive revenue with AI-powered conversations.

## What are Campaigns?

A **Campaign** is an automated outbound calling program that:

- Dials contacts from your list
- Connects answered calls to AI agents
- Handles voicemails and callbacks
- Tracks results and metrics

## Campaign Types

### Voice Campaigns

Outbound phone calls to prospects or customers:

| Use Case | Description |
|----------|-------------|
| **Lead Follow-up** | Contact new leads quickly |
| **Sales Outreach** | Reach prospects at scale |
| **Customer Check-ins** | Proactive customer success |
| **Event Reminders** | Appointment confirmations |
| **Re-engagement** | Win back inactive customers |

## Campaign Status

| Status | Description |
|--------|-------------|
| **Draft** | Campaign being configured |
| **Scheduled** | Waiting for start time |
| **Running** | Actively making calls |
| **Paused** | Temporarily stopped |
| **Completed** | All contacts processed |
| **Cancelled** | Permanently stopped |

## Key Metrics

| Metric | Description |
|--------|-------------|
| **Contacts** | Total numbers in campaign |
| **Attempted** | Calls dialed |
| **Connected** | Calls answered |
| **Connect Rate** | Connected / Attempted |
| **Conversions** | Desired outcomes achieved |
| **Conversion Rate** | Conversions / Connected |

## Best Practices

### Contact Lists

- **Clean data**: Verify phone numbers before upload
- **Segment wisely**: Group by relevance for better results
- **Honor opt-outs**: Respect contact preferences

### Timing

- **Respect time zones**: Call during appropriate hours
- **Optimal windows**: Typically 10am-12pm, 2pm-5pm local time
- **Avoid holidays**: Skip major holidays

### Pacing

- **Start slow**: Begin with lower volume to test
- **Scale gradually**: Increase after validating results
- **Monitor closely**: Watch for quality issues

## Compliance

<Warning>
  Ensure compliance with TCPA and local regulations. Compliance is your responsibility.

### Compliance Considerations

- Respect time zones and calling hours
- Honor opt-out requests
- Maintain proper consent records
