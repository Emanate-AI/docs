## Webhooks

Send data to custom endpoints:

```typescript
webhooks: [
  {
    event: "lead.created",
    url: "https://your-api.com/webhook",
    headers: { "X-API-Key": "{{API_KEY}}" }
  }
]
```

### Available Events

| Event | Trigger |
|-------|---------|
| `lead.created` | New lead captured |
| `lead.enriched` | Lead enrichment complete |
| `lead.qualified` | Lead meets ICP threshold |
| `call.ended` | Voice call completed |
| `chat.ended` | Chat session completed |
