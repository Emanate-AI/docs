## v2.4.0 - February 2024

### Highlights

This release focuses on **reliability improvements** for lead enrichment and Business DNA extraction, along with several quality-of-life enhancements.

### Updated Lead Enrichment

<Note>
  Lead enrichment is now more reliable with improved matching accuracy and better error handling.
</Note>

**What's New:**

- **Improved Email Matching**: Better handling of email variations and aliases
- **Company Domain Resolution**: More accurate company identification from email domains
- **Fallback Strategies**: Multiple matching strategies (email → phone → name+company)
- **Higher Match Rates**: 15% improvement in successful enrichment
- **Better Error Messages**: Clear feedback when enrichment fails

**Technical Details:**

```
Before: Single-pass email matching
After:  Multi-strategy matching with fallbacks

Match Priority:
1. Exact email match (highest confidence)
2. Domain + name match
3. Phone number match
4. Name + company fuzzy match
```

**Configuration:**

```typescript
enrichment: {
  provider: "apollo",
  strategies: ["email", "domain", "phone", "fuzzy"],
  minConfidence: 0.7,
  retryOnFailure: true
}
```

### More Reliable Business DNA Extraction

<Note>
  Business DNA extraction now handles edge cases better and provides more consistent results across different website types.
</Note>

**Improvements:**

- **Robust Logo Detection**: Better handling of SVG logos, inline logos, and various logo placement patterns
- **Enhanced Color Extraction**: More accurate brand color detection from logos, CSS, and theme colors
- **Improved Content Scraping**: Better handling of SPAs, lazy-loaded content, and bot-protected sites
- **Smarter GPT Synthesis**: More consistent field extraction with better fallback handling
- **Faster Extraction**: 30% reduction in average extraction time

**Logo Detection Improvements:**

| og:logo meta | ✓ | ✓ |
| JSON-LD schema | ✓ | ✓ |
| Header img detection | Basic | Enhanced |
| Inline SVG support | ✗ | ✓ |
| Base64 logo support | ✗ | ✓ |

**Color Extraction Improvements:**

| Source | Before | After |
|--------|--------|-------|
| theme-color meta | ✓ | ✓ |
| CSS variables | ✗ | ✓ |
| Logo pixel analysis | Basic | K-means clustering |

**Extraction Pipeline:**

```
URL → Scraper → Exa Deep Search → GPT Synthesis → Validation
       │           │                │
       │           │                └─ Better field mapping
       │           └─ 5 parallel calls for speed
       └─ Enhanced HTML parsing
```
