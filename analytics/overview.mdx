# Analytics Overview

Emanate provides comprehensive analytics to understand agent performance, conversation quality, and lead generation effectiveness.

## Dashboard

The main dashboard shows key metrics at a glance:

### Key Metrics

| Metric | Description |
|--------|-------------|
| **Total Conversations** | Calls + chats combined |
| **Connect Rate** | % of calls answered |
| **Avg Duration** | Average conversation length |
| **Leads Captured** | Successful lead captures |
| **Conversion Rate** | Leads / Conversations |

### Time Filters

- Today
- Last 7 days
- Last 30 days
- Custom range

## Filters

### Global Filters

Apply across all analytics:

- Date range
- Agent
- Channel (voice/chat)
- Outcome

### Saved Filters

Save common filter combinations:

```typescript
savedFilters: [
  { name: "High Value Leads", criteria: { icpScore: { min: 80 } } },
  { name: "This Week Voice", criteria: { channel: "voice", date: "this_week" } }
]
```
